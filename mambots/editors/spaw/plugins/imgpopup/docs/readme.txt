Image Popup Plugin for SPAW Editor PHP Edition v.2
--------------------------------------------------
This plugin allows users to insert image popups as links to the content.
User selects or uploads an image and link to popup dialog is created for that 
image.

Installation
------------
Just copy "imgpopup" directory into "plugins" subdir of your SPAW v.2 installation

Configuration
-------------
No configuration is necessary. However you can modify popup url and parameter
name if you want by changing appropriate values in config/config.php.

Toolbar button is added to "plugins" toolbar so it should be enabled for button
to be displayed (it is enabled by default)
