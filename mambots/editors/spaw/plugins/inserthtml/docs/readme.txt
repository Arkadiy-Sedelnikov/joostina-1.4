Insert HTML Plugin for SPAW Editor PHP Edition v.2
--------------------------------------------------
Allows users to paste arbitrary HTML code into current selection without having
to switch to HTML mode. This is especially useful when you get a piece of code
from sites like Flickr to insert into your page.

Installation
------------
Just copy "inserthtml" directory into "plugins" subdir of your SPAW v.2 
installation. 

Note: SPAW Editor PHP Edition v.2.0.1 or higher required

Configuration
-------------
No configuration is needed.
