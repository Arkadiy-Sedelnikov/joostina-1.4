CustomDropDown Plugin Example for SPAW Editor PHP Edition v.2
-------------------------------------------
This plugin adds a dropdown to plugins toolbar allowing user to select an item
and put the selected value directly on editing area.

Installation
------------
Just copy "customdropdown" directory into "plugins" subdir of your SPAW v.2 installation
