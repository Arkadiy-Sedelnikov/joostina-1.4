YouTube Plugin for SPAW Editor PHP Edition v.2
--------------------------------------------------
Embeds YouTube movie into the content based on YouTube page URL.

Installation
------------
Just copy "youtube" directory into "plugins" subdir of your SPAW v.2 
installation. 

Configuration
-------------
No configuration is needed.
