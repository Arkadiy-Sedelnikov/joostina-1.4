Change Case Plugin for SPAW Editor PHP Edition v.2
--------------------------------------------------
Adds 2 buttons to change case of selected text.

Installation
------------
Just copy "changecase" directory into "plugins" subdir of your SPAW v.2 
installation. SPAW Editor PHP Edition version 2.0.1 or higher required.

Configuration
-------------
No configuration is needed.
