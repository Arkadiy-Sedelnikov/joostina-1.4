Zoom Plugin for SPAW Editor PHP Edition v.2
-------------------------------------------
This plugin adds zoom dropdown to plugins toolbar allowing user zoom in and out
on editing area.

Supports Microsoft Ineternet Explorer Only.

Installation
------------
Just copy "zoom" directory into "plugins" subdir of your SPAW v.2 installation
